Given the matrix A = writeMatrix(MAA), determine A_{rref}.

matrixToText(rref(MAB))

To determine A_{rref}, we apply the algorithm found in Bretscher section 1.2:

For leading variable x_j, we find the first row (r_n) where x_j is non-zero.

For x_0 that is writeRow(MAA[pivot]). If there are rows where x_j is 0 above r_n, move r_n above them.

Then, we find the coefficient c of x_j in r_n.
For x_0 for example, c = C (because c \times 1 = x_j).

Now that we have c, we divide all elements in r_n by C, so that x_j equals 1:

writeMatrix(roundElements(divideC(MAA, pivot, 0)))

Then, we go over the rest of the rows and set x_j to 0 by subtracting multiples of r_n from each row.

To figure out how many times we should subtract r_n from an arbitrary row r_i, we use the element in r_i in the same column as x_j as a divisor d_i.

We then subtract r_n \times d_i from r_i.

After doing this, we get: writeMatrix(roundElements(subtractR(MAA,pivot,0), pivot, 0))

Repeat the last two steps for each r_i except r_n.

When all leading variables are processed, we get:
